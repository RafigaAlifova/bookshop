1. What is the GIT?
    Git yazdığımız proyektləri və ya programları internet üzərinde saxlamağa və idarə etməyə imkan verən nəzarət sistemidir
    
2. Why we use it?
    Çünki müəyyən məsələləri həll etməyi bizim üçün asanlaşdırırş Məsələn:
    Kompyuterdə problem yarananda yaddaşda olan kodlar silinə bilər, amma git kodlarımızı tam security şəkildə bizim üçün saxlayır
    Programların yazılma müddətində olan dəyişiklikləri izləməyə kömək edir.
    Bundan başqa deyə bilərik ki, git bir mənbə kodu üzərində bir neçə programçı işləyəndə də arada əlaqə qurur

3. What is the GitHub?
    Github üçün git program təminatı ilə əlaqələndirilmiş depo deyə bilərik.
    Müxtəlif git platformları var, biri də Githubdur. Yəni Githubun alternativləri var.

4. How the web browser works.
    Brauzer serverlə əlaqə yaradır.Sonra veb serverə HTTP sorğusu göndərir.
    Server sorğunu idarə edir və HTTP cavabını göndərir.
    Beləliklə brauzer veb səhifəni göstərir.

5. What are HTML tags for?
    Bir çox html tagi mövcuddur və deyə bilərik ki, fərqli məqsədlər üçün istifadə olunur.
    məsələn səhifənin başlıq (header) və ya sonunu (footer) təyin etmək üçün, elemnetləri qruplaşdırmaq üçün (div) 
    və ya sadəcə link (a),şəkil (img) ,adicə bir mətn(p) yeri yaratmaq üçün.
    taglar sadəcə 1 etiketdən də (input, a və s.)ibarət ola bilər, həm açılma həm bağlanma etiketindən də(img, div, body və s.) ibarət ola bilər.
6. What are Attributes for?
    Atributlar html elementləri haqqında əlavə məlumat verir
    Bütün elementlərin atributu ola bilər.
    Atributlar əsasən açılma tag'ində name="value" formasında yazılır, məsələn: href="index.html"

1. What does CSS do?
    CSS funksionallıq cəhətdən heç bir dəyişiklik etmir,
    sadəcə web saytlarımızın görünüşünü dəyişməyə və səliqəyə salmağa imkan verir.

2. What are CSS selectors for?
    Konkret olaraq desək,css kodlarımızı internal və ya external yazdığımız zaman görünüşünü dəyişmək istədiyimiz elementi(elementləri) seçməyə imkan verir.
    Veb səhifəmizin qurulusu bəzən murəkkəb olur, bu zaman css kodlarında elementləri sistemləşdirmək üçün selectorlar köməyimizə yetir.
